# opgave-8-frankstext

i denne opgave skulle jeg lave et program som tog teksten fra en bog, opg efter lavede nye sjove sætningerer med den, (se gerne billeder), her var noget af koden givet på forhånd og jeg skulle så lave den sidte del af koden, derfor laved jeg flowcharts til hver af de 
dele af koden jeg skulle lave.

den ene af mine koder, tjekker, bare om en token er upper, altså om den har stort forbogstav

den anden kode tjekker bare hvert tegn i teksten for ?!. og laver en return på true eller false,

et tredje kode Returnerer sidste tegn i en streng, eller '\0' hvis strengen er tom.

tildsist er der den fjere kode, som tjekker om noget er printbart, det kan være om det er tegn eller noget som den ikke kan printe, så skal den erstatte det med mellemrum
